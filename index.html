Projetos de Ciência de Dados

Desenvolvimento de um Dashboard Gerencial para Marketplace de Restaurantes com o Streamlit.

Neste projeto, foram empregados conceitos de Programação em Python, pensamento estratégico e de negócios, bem como manipulação e visualização de dados, a fim de criar um painel gerencial contendo as principais métricas para um Marketplace de restaurantes.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web,podendo ser acessado de qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy e Plotly
  • Anaconda e Jupyter Notebook.
  • Mapas interativos com Folium.
  • Framework Python Streamlit.
  • Streamlit Cloud.
  • Git e Github.
  • Saiba Mais (Github)
  • Demo (Streamlit)

Previsão de Vendas de Rede Farmacêutica utilizando Algoritmos de Regressão.

Nesse projeto de previsão de vendas para uma rede de farmácias, utilizou-se a linguagem Python e técnicas de Machine Learning. Foram coletados dados históricos de vendas e desenvolvido um modelo preditivo capaz de prever o faturamento para as próximas 6 semanas.

O produto de dados resultante foi hospedado na nuvem e disponibilizado através de um bot no aplicativo do Telegram,permitindo que os usuários obtenham as previsões de vendas de forma rápida e conveniente.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy e Seaborn.
  • Anaconda e Jupyter Notebook.
  • Algoritmos de Machine Learning com bibliotecas Sklearn, Scipy.
  • Técnicas de seleção de atributos.
  • Render Cloud, Flask e API Bot no Telegram.
  • Git e Github.
  • Saiba Mais (Github)
  • Demo (Youtube)

Classificar a Satisfação dos Clientes de uma Loja de vestuário utilizando Algoritmos de Classificação.

Nesse projeto de classificação de satisfação do cliente, utilizou-se a linguagem Python e técnicas de Machine Learning. A partir de dados históricos foi treinado um modelo de classificação capaz de classificar os clientes em 'Satisfeito' ou 'Neutro/Insatisfeito'.

A solução de dados resultante foi uma planilha no GoogleSheets contendo os clientes e seu estado de satisfação, disponível aos times para tomada de ações a fim reverter o cenário dos cliente insatifeitos.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy e Seaborn
  • Anaconda e Jupyter Notebook.
  • Algoritmos de Machine Learning com bibliotecas Sklearn,XGBoost e LightGBM.
  • Seleção de atributos utilizando a biblioteca Shap.
  • Ajuste de hiperparâmetros utilizando o framework Optuna.
  • Git e Github.
